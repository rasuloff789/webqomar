# International Qomar Quran Center Website

A modern, responsive website for the International Qomar Quran Center based in Istanbul, Turkey. This website provides information about online and offline Quranic education programs and allows students to register for courses.

## Features

- **Homepage**: Hero section, program overview, core values, and call-to-action
- **About Us**: Mission, educational programs, values, and location information
- **Registration**: Enrollment form with course selection and contact information
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations

## Technology Stack

- React 18
- React Router DOM
- Vite (build tool)
- CSS3 (custom styling)

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. Open your browser and navigate to `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The built files will be in the `dist` directory.

### Preview Production Build

```bash
npm run preview
```
